# Study Buddy 🎓

Study Buddy is a premium, AI-powered learning platform designed to help students master their course material through contextual document analysis, interactive chat, and automated study tools.

## 🚀 Vision
To create a seamless, aesthetically pleasing, and highly functional "digital brain" for students that turns passive reading into active learning.

## ✨ Core Features
- **📂 Universal Document Reader**: Support for PDF, PPTX, and Word documents with a high-fidelity viewing experience.
- **💬 Context-Aware AI Chat**: Chat with an AI tutor that knows exactly what you're looking at. "Explain this diagram" or "Summarize this page" in real-time.
- **🧠 Automated Study Tools**: Generate flashcards, Multiple Choice Questions (MCQs), and summaries directly from your course materials.
- **🏫 Course Management**: Organize your documents by college course. Keep your academic life structured and accessible.
- **☁️ Cloud Integration**: Pull documents directly from Google Drive or upload from your local machine.

## 🛠️ Technology Stack
- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Custom CSS Variables (Premium Aesthetic)
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **AI**: Gemini 2.0 / OpenAI (via custom API layer)
